损失函数

1.均方误差

MSE 通过平方差来强调较大的误差，使得模型在训练过程中更加关注减少大的预测误差。然而，MSE 对异常值比较敏感，因此在某些情况下，其他损失函数如平均绝对误差（Mean Absolute Error, MAE）可能更为合适。

2.交叉熵损失

交叉熵损失（Cross-Entropy Loss），也称为对数损失（Log Loss），是一种常用于分类问题的损失函数，特别是在二分类或多分类问题中。它衡量的是模型输出的概率分布与真实标签的概率分布之间的差异。

交叉熵损失的核心思想是，如果模型的预测概率分布与真实的标签概率分布越接近，损失就越小，模型的性能就越好。在训练过程中，通过优化算法（如梯度下降）来调整模型的参数，以最小化交叉熵损失，从而提高模型的预测准确性。

优化算法

在机器学习和深度学习中，优化器是用于调整模型参数以最小化损失函数的算法。下面是几种常见的优化器的详细解释：

    • 最基本的优化器，通过计算损失函数对每个参数的梯度，然后更新参数以减少损失。

  2. Momentum：

    • 在SGD的基础上，引入了动量（momentum）的概念，帮助加速SGD在正确方向上的进展，并减少震荡。

  3. AdaGrad：

    • 自适应梯度算法，为每个参数调整学习率，对于频繁更新的参数减少学习率，对于不常更新的参数增加学习率。
    • 更新规则：
    • ( G ) 是累积的梯度平方和，(\epsilon) 是一个小常数以避免除以零。

  4. RMSprop：

    • 解决了AdaGrad中学习率持续下降的问题，通过指数衰减的方式来计算平方梯度的滑动平均。
    • 更新规则：
    • ( G ) 是平方梯度的滑动平均。

  5. Adam：

    • 结合了Momentum和RMSprop的优点，同时考虑了梯度的一阶矩（均值）和二阶矩（方差）。

每种优化器都有其特定的适用场景和优缺点。例如，SGD简单但可能需要较长时间收敛，而Adam通常能够更快地收敛但可能需要仔细调整超参数。选择哪种优化器取决于具体问题和模型的复杂性。在实际应用中，可能需要尝试不同的优化器来找到最适合特定任务的算法。
